# Career Radar

`career_radar.jsx` — an interactive Recharts tool scoring career/market scenarios
(VN, SG, global-remote, etc.) across six dimensions: lifestyle fit, cost
efficiency, take-home pay, career upside, domain fit, and sustainability.

Use it to drive the direction decision that feeds `../goals-and-targets.md` and the
market emphasis in `../positioning-thesis.md`.

Single-file React component — drop into any Vite/CRA sandbox with `recharts`
installed to run.
